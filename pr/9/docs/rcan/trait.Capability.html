Expand description

A trait for types that define a capability.

Capabilities can be compared using Capability::permits, which determines whether one capability grants permission to perform another.

A common implementation of this trait might be an enum representing different RPC request types.

The Capability type must be serializable so it can be included in the signature payload in an Rcan.
